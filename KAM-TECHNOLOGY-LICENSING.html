1. Executive Summary

Total NPM Packages Used: 50+ packages

Open Source Packages (MIT/Apache): 100% of frontend packages

Packages Requiring License: 0 (all frontend packages are free)

Potential Costs: SQL Server (if exceeding Express limits), Hosting infrastructure

The ParkBuddy is built using modern web technologies with a focus on open-source solutions. All frontend libraries and frameworks used are licensed under permissive open-source licenses (MIT or Apache 2.0), which allow free commercial use without royalty payments.
